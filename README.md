# expressjs-gaming-api
A simple RESTful API for managing video games, built with Express.js and Prisma.  
⚠️ **Work in Progress** – this project is under active development.

This project demonstrates basic CRUD operations, structured routing, and proper API response handling. It serves as a starting point for building a more complete gaming API.

## Tech Stack
* Node.js
* Express.js
* TypeScript
* Prisma
* SQLite
* REST

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Generate Prisma client: `npx prisma generate`
4. Run the dev server: `npm run dev`
